    阅读论文

    一篇期刊论文，主要分成四个部分。

    • Abstract：说明这篇论文的主要贡献、方法特色与主要内容。学会只看 Abstract 和 Introduction 便可以判断出这篇论文的重点和你的研究有没有直接关连。

    • Introduction：介绍问题的背景和起源，交代前人在这个题目上已经有过的主要贡献，说清楚前人留下来的未解问题，以及在这个背景下这篇论文的想解决的问题和它的重要性。收集跟题目可能相关的论文 30～40 篇，只读 Abstract 和 Introduction，稍微参考 Illustrative examples 和 Conclusions，回答：

      1. 在这领域内最常被引述的方法有哪些？
        • 从相关的数篇 review paper 开始，从中根据 title 与 Abstract 找出你认为跟你研究题目较相关的 30～40 篇论文。
      2. 这些方法可以分成哪些主要派别？（按日期排序）
      3. 每个派别的主要的诉求、方法特色和优点是什么？每个派别的缺点是什么？（通常是其他论文写的，观察对其他派别的批评加入缺点）
      4. （根据上面一问）这个领域内大家认为重要的关键问题有哪些？有哪些特性是大家重视的优点？有哪些特性是大家在意的缺点？这些优点与缺点通常在哪些应用场合时会比较被重视？在哪些应用场合时比较不会被重视？
      5. 将各个派别的论文整理在同一个档案夹里，按时间顺序排好：

    • Main body（含 simulation and/or experimental examples）： 在你第一次有系统地念某派别的论文 main bodies 时，你只需要念懂：

      1. 这篇论文 的主要假设是什么（在什么条件下它是有效的），并且评估一下这些假设在现实条件下有多容易（或多难）成立。愈难成立的假设，愈不好用，参考价值也愈低。
      2. 在这些假设下， 这篇论文主要有什么好处。
      3. 这些好处主要表现在哪些公式的哪些项目的简化上。
      4. 根据方法缺点和方法需要的假设回答：这一派主要的缺点有哪些。
        • 参考 Examples：当某某假设在某些实用场合上无法成立时，这个方法会不会出什么状况？猜一猜，预测一下这个方法应该会在哪些条件 下（应用场合）表现优异，又会在哪些条件下（应用场合）出状况？

    其他

    • 不需要理解恒等式推导过程。
