    排序

    整理：2024 年 1 月 7 日 于上海交通大学

    前置代码

    #define MAXSIZE 10000   /* 用于要排序数组个数最大值，可根据需要修改 */
typedef struct
{
    int r[MAXSIZE+1];   /* 用于存储要排序数组，r[0]用作哨兵或临时变量 */
    int length;         /* 用于记录顺序表的长度 */
}SqList;


/* 交换L中数组r的下标为i和j的值 */
void swap(SqList *L,int i,int j) 
{ 
    int temp=L->r[i]; 
    L->r[i]=L->r[j]; 
    L->r[j]=temp; 
}

    ⚠️注意：下方代码下标都是从 1 开始的

    冒泡排序

    冒泡排序是一种交换排序，两两比较相邻记录的关键字，如果反序则交换，直到没有反序的记录为止。

    个人感觉从后往前循环，如同较小值慢慢从底部浮上来会使这个算法更加“名副其实”一点。

    void BubbleSort(SqList *L)
{
    int i, j;
    for(i=1; i < L->length; i++)
    {
        for(j=L->length-1; j>=i; j--) // 从最后往前循环直到 i
        // for(j=1; j < L->length-i; i++)，也可以从开头向后循环直到 length-i，因为之后的位置已经被归位
        {
            if(L->r[j] > L->r[j+1]) // 前者大于后者
            {
                swap(L, j, j+1);
            }
        }
    }
}

    复杂度： - 最坏的情况：需要比较 \(\frac{n(n-1)}{2}\) 次，并做等数量的交换 总的时间复杂度为 \(O(n^ 2)\)

    选择排序

    简单选择排序法：在现有的记录中，不断选择最小值、第二小值……依次与对应的位置交换

    void SelectSort(SqList *L)
{
    int i, j, min;
    for(i=1; i < L->length; i++)
    {
        // 在余下的记录中寻找最小值
        min = i;
        for (j=i+1; j <= L->length; j++)
        {
            if (L->r[min] > L->r[j])
                min = j;
        }
        if (i != min)
            swap(L, i, min);
    }
}

    复杂度： - 比较次数，都需要比较 \(\frac{n(n-1)}{2}\) 次 - 交换次数，最好的时候为 0，最差的时候为 \(n-1\) 次 总的时间复杂度为 \(O(n^ 2)\)

    直接插入排序

    直接插入排序的基本操作是将一个记录插入到已经排好序的有序表中，从而得到一个新的、记录数增 1 的有序表。

    void InsertSort(SqList *L)
{
    int i, j;
    for (i=2; i <= L->length; i++)
    {
        // 需要将 L->r[i] 插入有序子表，假设 1~(i-1) 都排列好
        if (L->r[i] < L->r[i-1]) 
        {
            L->r[0] = L->r[i]; // 设置哨兵 
            for(j=i-1; L->r[j] > L->r[0]; j--)
                L->r[j+1] = L->r[j]; // 将元素前移，并且需要一一交换
            L->r[j+1] = L->[0]; // 停止
        }
    }
}

    时间复杂度：

    • 最好的情况：if (L->r[i] < L->r[i-1]) 判断，\(O(n)\)
    • 最坏的情况：\(\sum_{i=2}^n = (n+2)(n-1) / 2\)，即 \(O(n ^ 2)\)

    快速排序

    快速排序：通过一趟排序将待排记录分割成独立的两部分，其中一部分记录的关键字均比另一部分记录的关键字小（大），则分别对这两部分记录继续进行排序，以达成有序的目的。

    由于需要递归调用，所以外封装一个函数。 

    /* 对顺序表L作快速排序 */
void QuickSort(SqList *L)
{ 
    QSort(L,1,L->length);
}
    对顺序表中的子数列进行快速排序：（1）分割（2）对低子表递归排序（3）对高子表递归排序 
    /* 对顺序表L中的子序列L->r[low..high]作快速排序 */
void QSort(SqList *L,int low,int high)
{ 
    int pivot;
    if(low<high)
    {
        /* 将L->r[low..high]一分为二，算出枢轴值pivot */
        pivot=Partition(L,low,high); 
        QSort(L,low,pivot-1);        /*  对低子表递归排序 */
        QSort(L,pivot+1,high);       /*  对高子表递归排序 */
    }
}

    Partition 函数需要选取当前的关键字，然后想办法把它放到一个位置，使它左边的值比它小，右边的值比它大。这样的关键字命名为枢轴 (pivot) 并且待确定。实现方法是：

    int Partition(SqList *L,int low,int high)
{/* 交换顺序表L中子表的记录，使枢轴记录到位，并返回其所在位置，此时在它之前(后)均不大(小)于它。*/
    int pivotkey;

    pivotkey=L->r[low];     /* 用子表的第一个记录作枢轴记录 */
    while(low<high)         /* 从表的两端交替地向中间扫描 */
    { 
         while(low<high && L->r[high] >= pivotkey)
            high--;
         swap(L,low,high);  /* 将比枢轴记录小的记录交换到低端 */
         while(low<high && L->r[low]<=pivotkey)
            low++;
         swap(L,low,high);  /* 将比枢轴记录大的记录交换到高端 */
    }
    return low;             /* 返回枢轴所在位置 */
}

    或是同时在左右边找符合各自条件的，当两者同时被找到时交换；直至 low>high
