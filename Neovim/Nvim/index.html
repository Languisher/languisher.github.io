    配置 Nvim

    毫不夸张的说，我在配置一个好看的 Nvim 窗口上花费了巨量功夫，配置固然越来越好看，但消耗了很多精力且做了很多重复且无用功，整个过程还是比较沮丧，无论是阅读各种文档还是观看各种教学博客和视频。最先我先用 Vim-Plug，在偶然接触到 LazyVim 之后就爱上了这个由各式 .lua 文件组成的配置。

    先后一共有四版，最后还是采用了由 LazyVim 官方教程和加上我个人一些自定义配置的版本。每一版均能正常运行，没有优劣之分。

    第三版配置

    1. Install LazyVim Starter
    2. Look for the keymaps introduction for more advice.

    第二版配置

    My configuration of nvim can be download here (Github link).

    Open the cheatsheet of NvChad and search for commands : <leader>ch

    NvChad

    Visit the doc of Nvchad.

    Themes

    Change the theme: <leader>th

    LazyNvim - Plugin Manager

    Use :Lazy to open LazyNvim UI.

    Add/Make changes to a plugin: (This part is different from the official documentation) - Add the plugin spec to the file lua/custom/plugins.lua. - Visit the doc

    Example:

    local plugins = {
  ...
  { "ellisonleao/gruvbox.nvim" },
  ...
}

return plugins

    Tabufline

    • Goto prev/next buffer: <S-Tab>, <Tab>
    • Close the buffer: <leader>x

    Telescope

    • Find ... <leader>f and wait for the suggestions to pop out.

    Nvim-Tree

    Type g? to see the mappings

    General operations: - Quit Nvim-Tree: q - Expand/Collapse all: E/W - Toggle Filter: Dotfiles: H

    File related: - Copy relative/absolute path: Y/gy - Run system with default setting: s - Copy/Cut/Delete/Rename: c, x, d, r - Jump to parent dir: P
